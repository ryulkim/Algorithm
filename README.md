# Algorithm
코딩테스트 목적 algorithm 스터디  
> 참여자: 김률아, 소연수, 한다연

## 문제사이트
프로그래머스 고득점 Kit  
-> https://programmers.co.kr/learn/challenges  


## 스터디 방식
* 일주일 1번(일) 주제별 발표  
* 발표전까지 주제에 맞는 문제 풀어오기

## 스터디 주제

> ### 1주차
> * Hash(해시)
> * 발표자: 김률아  
> ### 2주차
> * Sorting(정렬)
> * 발표자: 한다연  
> ### 3주차
> * BruteForce(완전탐색)
> * 발표자: 소연수  
> ### 4주차
> * DFS-BFS(깊이/너비 우선 탐색)
> * 발표자: 한다연
